# Quick Add Template Notes

## What Changed

- Replaced the previous Habit Blocks with workflow-specific templates that match Iris’s actual study system.
- Renamed **AI Project** to **AI Coding**.
- Split **English** into:
  - **English Output** for daytime active speaking/shadowing/reuse work.
  - **English Input** for lower-pressure evening listening, reading, and NotebookLM summaries.
- Added **Japanese** as a light language-maintenance and identity category.
- Moved **Cyber** later in the list so the homepage does not open with the highest-stress study area.
- Added **Job / Career** as its own category so career work no longer lives inside Admin / Life.
- Added a dedicated **Review / NotebookLM** category for low-input useful work.
- Added notes to every template so newly created blocks are specific enough to start immediately.
- Added subtasks to:
  - Shadowing full cycle.
  - Oral summary practice.
- Added subtle evening and rescue guidance without hiding any templates.

## Why Cyber Moved Later

Cyber is important, but it has a higher emotional load because it is tied to assessments, labs, evidence, and deadlines. Moving it later keeps the homepage from feeling stressful immediately while still keeping Cyber available in one tap.

## Why Japanese Was Added

Japanese is a light maintenance habit and part of Iris’s language identity. It should be easy to add on low-energy evenings without being mixed into English or admin work.

## Why Job / Career Was Added

Career work has its own emotional load and purpose: CV, cover letter, LinkedIn, interviews, selection criteria, portfolio writing, and positioning. Keeping it separate stops Admin / Life from becoming a messy bucket and makes job work easier to choose deliberately.

## Job / Career Energy Guide

Low-energy friendly:

- Job search review.
- LinkedIn profile polish.
- Career positioning block.

Daytime recommended:

- Cover letter draft.
- Interview practice.
- CV update block.
- Selection criteria / STAR answer.
- Portfolio / project write-up.

## Behavior

- Clicking a template adds a Day Block to today’s Block Queue.
- The created block includes title, type, area, priority, estimate, energy level, notes, and any predefined subtasks.
- After 17:00, high-output or deep-work templates are marked **daytime recommended**.
- In Rescue Day mode, easiest useful templates are marked **rescue-friendly**.

Evening mode naturally favours lower-input categories:

- English Input
- Japanese
- Job / Career, when choosing low-pressure review or positioning blocks
- Review / NotebookLM
- Admin / Life
- Reset

High-output or high-stress categories remain available, but are visually marked when they are better suited to daytime.

## Known Limitations

- Quick Add templates create Block Queue items only; they do not create Task Inbox records.
- Evening mode only labels high-output templates; it does not block manual selection.
- Rescue highlighting is heuristic and may need tuning after real use.
