# Media Tab Notes

## What changed
- Added a top-level Media tab for a light personal media log.
- Added quick links to Iris Book Finder and Expression Review Hub.
- Added localStorage persistence under `iris-media-log`.
- Added filters for English, Japanese, Audiobook, TV, Podcast, Useful for Shadowing, and Low-angst.
- Added starter recommendation pools for English/shadowing, low-angst comfort media, and Japanese maintenance.

## Obsidian
- Each media entry can copy an Obsidian-style media note.
- Suggested destination changes by language:
  - English / mixed: `01 English 英语/Media Notes/`
  - Japanese: `02 Japanese 日语/Media Notes/`

## Scope
- This is intentionally not a heavy database.
- Media-to-study integration starts with copy actions so Study Dashboard and existing sessions remain untouched.
