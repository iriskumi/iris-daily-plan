# Media Tab Beige Typography Polish

## What Changed

- Replaced the heavier add-entry prompt with the softer title `记录一个素材`.
- Added the helper line `书、剧、播客、有声书、精听片段都可以。`.
- Added lighter helper text to Quick Links, Recommended Pools, and Media Log.
- Scoped Media tab styling under `.media-tab-page` so the beige palette does not affect Today, Study, Plan, Exercise, or Integrations.

## Visual Direction

- Shifted Media tab surfaces toward milk foam, vanilla, almond, and warm beige.
- Reduced pink/peach emphasis in cards, chips, borders, and active states.
- Restyled form inputs as soft notebook fields with almond borders and cocoa text.
- Restyled buttons and links as warm cocoa/beige controls instead of bright pink CTAs.
- Media log cards now read more like quiet index cards.

## Typography

- Kept the light descriptive subtitle feel for helper text.
- Applied the softer Chinese font stack only to Chinese helper/prompt copy.
- Kept functional UI labels, inputs, and controls readable and unchanged in behavior.

## Scope

- No app logic changed.
- No localStorage/data shape changed.
- Media entry creation, filters, quick links, Obsidian copy, and existing entries continue to use the same data flow.
